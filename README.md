# Janiyah-GPT

An AI Literacy, Prompt Engineering, Academic, and Career Coaching
Co-Pilot developed as part of the Black-Liberation.Tech ecosystem.

## Purpose

[Janiyah GPT](https://chatgpt.com/g/g-68fe10a0c2cc819189535e83ddfd6b5e-janiyah) helps learners connect AI literacy with identity,
reflection, portfolio development, career exploration,
and ethical technology use.

## Focus Areas

• Career Discovery
• Black-Liberation.Tech
• Prompt Engineering
• Reflection
• Portfolio Building
• Responsible AI
